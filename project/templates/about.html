Aufgabe B-1 Web-Anwendung mit Flask
In dieser Aufgaben sollen Sie eine Web-Anwendung mit Flask entwickeln. Sie brauchen dazu die Python-Pakete flask, flask-wtf, und SQLALchemy. Diese können Sie durch die KonsoleBefehle pip install flask, pip intall flask-wtf bzw. pip install SQLAlchemy installieren. Verwenden Sie die Datenbank sqlite. Ein SQLAlchemy-Tutorial finden Sie unter http: http://docs.sqlalchemy.org/en/latest/orm/tutorial.html.

a) Entwickeln Sie eine Web-Anwendung, die eine persönliche TODO-Liste verwaltet. Eine TODO-Liste besteht aus TODO-Items. Ein TODO-Item hat ein Feld due, das in Ihrer Anwendung als String dargestellt werden kann, und die Felder title und description, die Strings sind. Desweiteren hat das TODO-Item auch einen state, der entweder new oder finished ist.

b) Die Anwendung soll folgende Funktionalitäten bieten
  • • alle TODO-Items anzeigen
  • • ein neues TODO-Item anlegen
  • • den Zustand eines TODO-Items ändern
  • • nach TODO-Items suchen
  • • ein TODO-Item aus der DB löschen


c) Sorgen Sie mit geeigneten HTML- und CSS-Angaben fur eine akzeptable GUI

d) Kommentieren Sie Ihr Python-Programm hinreichend

e) Modellieren Sie in einem UML-Klassendiagramm die Klassen TODOList, TODOItem, und deren Beziehungen.
